Implementation of heap in C++ and Haskell.

## C++

### Compile

`$ g++ -o heapsort heapsort.cpp`

### Run

`$ ./heapsort`

## Haskell

### Run

* Load `ghci`

`$ ghci`

* Load `heapsort.hs` module

`Prelude> :l heapsort.hs`

* Call heapsort on a list

`*Main> heapsort <list>`
